# PhotoPrism - Photo Management

Photo management with AI/ML features (Google Photos alternative).
Set up for local A/B testing against Immich.

## Features

- AI-powered tagging and categorization
- Face recognition
- Map view with GPS data
- RAW file support
- Full-text search
- WebDAV support

## Quick Start

```bash
just up photoprism
```

Access at `https://photoprism.${BASE_DOMAIN}`.
