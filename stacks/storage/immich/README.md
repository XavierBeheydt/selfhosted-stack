# Immich - Photo Management

Photo management with ML-powered features (Google Photos alternative).
Set up for local A/B testing against PhotoPrism.

## Features

- Mobile app with auto-backup
- Face detection and recognition
- Object and scene detection
- Geolocation and timeline
- Sharing and albums
- HEIC/RAW support

## Quick Start

```bash
just up immich
```

Access at `https://photos.${BASE_DOMAIN}`.
