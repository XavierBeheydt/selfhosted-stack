# n8n - Workflow Automation

Low-code workflow automation with visual editor.

## Features

- Visual workflow builder
- 400+ integrations
- Webhook triggers
- Custom code nodes (JavaScript/Python)
- Used for Meilisearch data indexing connectors

## Quick Start

```bash
just up n8n
```

Access at `https://n8n.<BASE_DOMAIN>`.

## Data Connectors

n8n runs the indexing workflows that feed data into Meilisearch:
- Dolibarr invoices, contacts, proposals
- Seafile file metadata
- Immich/PhotoPrism photo metadata
