# Ollama - LLM Inference

> **Status**: Phase 3 - Not yet implemented

[Ollama](https://ollama.ai/) provides local LLM inference with GPU acceleration.

## Planned Configuration

- RTX 4090 GPU passthrough (NVIDIA Container Toolkit)
- Model storage on local SSD
- API exposed to Open WebUI and n8n
- Deployed on local PC (GPU node) with `nodeSelector`
