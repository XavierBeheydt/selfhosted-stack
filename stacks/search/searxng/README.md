# SearXNG - Meta Search Engine

Privacy-first meta search engine aggregating results from 70+ sources.

## Features

- Privacy-respecting (no tracking, no ads)
- 70+ search engine sources
- Privacy engines enabled by default (DuckDuckGo, Brave, Startpage)
- Google/Bing available but disabled by default
- JSON API for integration with Open WebUI RAG

## Quick Start

```bash
just up searxng
```

Access at `https://sx.${BASE_DOMAIN}`.

## Configuration

Edit `config/settings.yml` to enable/disable engines and tune defaults.
